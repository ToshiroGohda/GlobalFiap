# ENDPOINTS

<h2>/usuario</h2>
  
  POST E GET ALL: /usuario

  GET BY EMAIL: /usuario/email/{email}
  
  GET BY ID: /usuario/id/{id}

  PUT E DELETE: /usuario/{id}
<br>
<b2>/publicacao</h2>

  POST E GET ALL: /publicacao
  
  PUT, DELETE AND GET BY ID: /publicacao/{id}
